# substrait-rs

Rust bindings for [substrait](https://substrait.io/).

## Build & Test

Clone this repository.

```bash
git clone git@github.com:substrait/substrait-rs.git
cd substrait-rs
```

Update submodules.

```bash
git submodule init
git submodule update
```

Build and test with Cargo.

```bash
cargo test
```
